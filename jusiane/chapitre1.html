Chapitre I : Le Bureau de Lord Arthur

Le corps sans vie de Lord Arthur gît sur son fauteuil. La police a bouclé le manoir et retient les 4 suspects principaux présents ce soir-là :

• Elena (l'épouse)
• Nestor (le demi-frère)
• Dr. Moretti (le médecin)
• Victor (le majordome)

En fouillant la pièce, tu remarques un double fond dissimulé sous le sous-main du bureau. Un coffre-fort numérique y est encastré. Posé juste à côté, un fragment déchiré du testament d'Arthur contient la clé pour trouver le code.

Le fragment du testament :
"À mon demi-frère Nestor, en raison de ses dettes massives, je ne laisse strictement RIEN. L'intégralité de ma fortune reviendra à..."

L'énigme du clavier numérique :
Une note manuscrite d'Arthur indique la logique du code à 4 chiffres :

1. 1er chiffre : Le nombre de suspects restants si on écarte Nestor.
2. 2e chiffre : Le nombre total de lettres du prénom de la victime (ARTHUR).
3. 3e chiffre : Le double du premier chiffre.
4. 4e chiffre : Le nombre total de suspects au départ.
📌 Regarde la liste au-dessus :
- Total suspects : 4 (donc 4e chiffre = 4).
- Sans Nestor, il en reste : 3 (donc 1er chiffre = 3).
- Lettres d'A-R-T-H-U-R : 6 (2e chiffre = 6).
- Le double du premier (3 x 2) : 6 (3e chiffre = 6).
Le code est donc : 3664.

Entrez le code à 4 chiffres :

Code incorrect. Le coffre-fort reste verrouillé.
